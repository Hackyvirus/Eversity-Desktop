5) Property: Box Model
Height: vertically Area
Width: Horizontally Area
Padding: content to border
border: border of content
Margin: border outer Area

6) Property: Display
Inline: element becomes Inline element
Block: element becomes block element
Inline-block: element becomes Inline element but we can give Height and Width
none: remove element from document
visibility: hidden


7) Property: Units is CSS

A) Absosule Units
1px
1in
1cm
B) Relative Units
1%: 1 % of parent value
1em: relative to parent element
1rem: relative to root element
1vh: relative to Height of the Display
1vw: relative to width of the Display

8) Property: Position
static: default Position of element in document
relative: Position of the elemet relative to its actual Position
Absosule: Position of element relative to browser window
fixed: Position fixed at perticular Position
sticky: Position sticky to a Position
z-index: Position of element acording to z axis
